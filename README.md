# Augmented Planet Explorer

## Beschreibung
Die App Augmented Planet Explorer verwendet Augmented-Reality-Technologie, um eine interaktive Erkundungserfahrung der Welt um uns herum zu bieten. Nutzer können durch ihre Smartphones auf verschiedene Orte zugreifen und detaillierte Informationen über die Geschichte, Kultur und Natur erfahren.

## Funktionen
- **Interaktive 3D-Modelle**: Sehen Sie historische Stätten und Naturwunder in 3D.
- **Informative Inhalte**: Erfahren Sie mehr über die Bedeutung jedes Ortes durch interessante Fakten und Anekdoten.
- **Entdeckung in der Nähe**: Finden Sie Sehenswürdigkeiten in Ihrer unmittelbaren Umgebung.
- **Benutzerinteraktionen**: Nutzer können Inhalte liken, kommentieren und eigene Erlebnisse teilen.

## Installation
1. Klonen Sie dieses Repository.
2. Stellen Sie sicher, dass Sie Node.js installiert haben.
3. Führen Sie `npm install` im Hauptverzeichnis aus.
4. Starten Sie die App mit `npm start`.

## Technologien
- React Native für die App-Entwicklung
- ARKit/ARCore für Augmented-Reality-Funktionen
- MongoDB für die Datenspeicherung von Inhalten
